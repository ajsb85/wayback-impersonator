wayback

A concurrent, browser-impersonating scraper in Rust to download archived assets from the Internet Archive's Wayback Machine CDX API without being blocked.

Install via APT Repository

To keep your installation updated automatically, you can add this repository directly to your system's package manager:

# 1. Download and trust the repository GPG key
sudo wget -O /usr/share/keyrings/wayback-keyring.gpg https://ajsb85.github.io/wayback-impersonator/amd64/archive-key.gpg
# 2. Add the repository to your APT sources
echo "deb [signed-by=/usr/share/keyrings/wayback-keyring.gpg] https://ajsb85.github.io/wayback-impersonator/amd64/ ./" | sudo tee /etc/apt/sources.list.d/wayback.list
# 3. Update package index and install wayback
sudo apt update && sudo apt install wayback

Prerequisite Warning

wayback requires libcurl-impersonate at runtime to mimic browser handshakes. Please make sure the library is installed and discoverable on your system. Refer to the project's README for the quick installation guide.
